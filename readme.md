# Untitled Space Game

Game created for the Js13kGames competition from 2021.

## NPM Scripts

| Npm Run | Action                                          |
| ------- | ----------------------------------------------- |
| dev     | Start the game in development mode.             |
| build   | Compile, compress and zip the game for release. |
